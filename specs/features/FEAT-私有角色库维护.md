# 私有角色库维护

- 状态：approved
- Spec ID：`SPEC-ROLELIB`

## 功能描述

本文件统一定义私有角色库的定位、维护工具、校验规则和入口层级。

私有角色库解决的是“模型不认识冷门角色或私有对象，导致无字图难搜”的问题。它是高级检索增强能力，不是普通用户主路径，也不是通用设置页的一部分。

## 适用对象

- 项目维护人
- 高级用户
- 需要为少量冷门角色建立稳定召回锚点的人

## 维护内容

- 角色名与别名
- 示例图片
- 必要备注
- 当前规则下的可用性校验和测试

维护工具应优先暴露少量核心字段，不继续外露旧知识库时代的复杂兼容字段和调参语义。

## 入口层级

- 不作为普通用户一级导航
- 当前实现仅在开发模式下通过图库管理的高级能力区、开发模式页面或受控入口进入
- 对外文案要强调“冷门角色 / 私有对象增强”，而不是抽象的知识库概念

## 与搜索主路的关系

- 只有当查询明显包含角色主体线索时，才适合触发私有角色增强
- 私有角色增强应先限定主体，再配合动作、表情或场景细排
- 角色库命中失败时，系统应能降级回 OCR / 语义主路，而不是把搜索彻底带偏

## 不负责

- 不承担普通图片标签体系
- 不面向普通用户做大规模内容运营
- 不替代搜索主路的基础 OCR 和语义能力

## 验收要点

- 维护人可以创建、编辑、校验和测试少量私有角色卡片
- 角色卡片字段足够少，维护成本可控
- 私有角色库不会误入普通用户主导航
- 运行时命中和维护工具校验使用同一套语义
- 当前实现中，维护路由 `/private-role-maintenance` 只在开发模式注册
