# 图库管理模块

- 状态：approved

## 模块职责

图库管理模块负责：

- 导入后的补充治理
- 批量追加图片
- 查看最近新增
- 查看异常图片
- 清理失效图片、删除图片、补充信息

## 不负责

- 不替代搜索页承接按记忆线索找图
- 不作为首次导入默认必经页面
- 不承载底层检索权重调参

## 当前结构

- 一级页面：`图库管理`
- 视图：`全部图片 / 最近新增 / 异常图片`
- 导入入口：`添加图片 / 添加文件夹`
- 失败修复入口来自搜索页
