# 图库管理模块

- 状态：approved

## 模块职责

图库管理模块负责：

- 导入后的补充治理
- 批量追加图片
- 查看最近新增
- 查看异常图片
- 清理失效图片、删除图片、补充信息

## 不负责

- 不替代搜索页承接按记忆线索找图
- 不作为首次导入默认必经页面
- 不承载底层检索权重调参

## 当前结构

- 一级页面：`图库管理`
- 视图：`全部图片 / 最近新增 / 异常图片`
- 导入入口：`添加图片 / 添加文件夹`
- 失败修复入口来自搜索页

## 导入状态与跨界面联动

这部分统一引用状态机文档，不再在模块文档里重复展开长说明。

### 当前口径

- 导入、恢复、角标、弹层、图库页顶部承接统一遵循 [`import-state-machine.md`](../architecture/import-state-machine.md)
- 任何修改首页图库按钮、启动恢复弹层、图库页顶部横幅/进度/结果摘要、任务队列与恢复命令的改动，都应先对照这张状态机

### 最小检查点

后续改动至少确认下面 5 个问题仍然成立：

1. 现在是不是还在处理？
2. 还剩多少没处理？
3. 我现在需不需要立刻做决定？
4. 刚才继续导入后结果怎么样？
5. 这个提示什么时候该消失？
