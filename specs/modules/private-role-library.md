# 私有角色库模块

- 状态：approved

## 模块职责

私有角色库负责：

- 面向高级用户或维护人的小规模角色增强维护
- 为冷门角色或私有对象提供更稳定的主体识别锚点
- 提供维护、校验与测试能力

私有角色库不负责：

- `match_terms` 或其他补充搜索词维护
- 基于 OCR / 文件名的自动角色打标
- 动作、表情、场景等非主体线索建模

## 对普通用户的约束

- 不作为普通用户一级导航
- 不作为搜索主路径的必经概念
- 可作为图库管理中的低干扰正式入口存在，但不进入普通用户一级导航

## 当前访问定位

- `图库管理` 中的轻量角色增强入口
- 搜索失败后的受控引导入口
- 同仓库内工具或 CLI
