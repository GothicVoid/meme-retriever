# Feature Spec 说明

当前项目仍沿用历史 `5.x.x` 编号体系，但当前有效正文已收口到 `product/features/`。

当前结构说明：

- `product/features/` 存放当前有效的 feature 正文
- `specs/` 根目录中的多数 `5.x.x` 文档为兼容入口，仅保留跳转说明
- 少量根目录文档仍保留为总览入口，例如 `5_3_2.md`
- `product/modules/` 提供当前长期模块口径
- `changes/` 提供新旧口径切换说明

阅读时若同时看到 `FEAT-...` 文档和根目录同编号兼容入口，应以前者为准。
