#### 功能描述
以全屏或居中大图方式展示图片详细信息，**GIF格式自动播放动画**，支持快速操作。

#### 流程说明
1. 用户双击缩略图或右键"查看详情"
2. 打开详情页（模态框形式）
3. **如果是GIF格式，自动播放动画**
4. 展示图片元数据（尺寸、格式、大小等）
5. 提供快捷操作按钮（复制、删除、编辑标签等）
6. 按Esc或点击关闭按钮退出
7. 文件丢失时显示异常态，支持重新定位或直接删除记录

#### 交互说明
**页面元素：**
- 返回按钮：左上角，点击关闭详情页
- 关闭按钮：右上角"✕"
- 图片展示区：居中显示，**GIF自动循环播放**
- 文件信息卡片：显示元数据
- 标签卡片：显示和编辑标签
- 操作按钮组：底部固定

**交互行为：**
- **GIF动画控制：**
  - 自动播放：打开详情页后立即播放
  - 循环播放：无限循环
  - 暂停/播放：鼠标悬停在GIF上时显示控制按钮
  - 播放速度：原始帧率播放
- 点击图片外区域：关闭详情页
- 按Esc：关闭详情页
- 左右方向键：切换上/下一张图片（如果在检索结果中）
- 点击"复制"：复制图片到剪贴板
- 点击"在文件夹中显示"：打开系统文件管理器并定位到该文件
- 点击标签"+"：弹出标签输入框
- 文件丢失时：
  - 禁用标签保存等依赖原文件的操作
  - 显示"重新定位"按钮，允许用户恢复记录
  - 显示"删除图片"按钮，允许用户直接删除该记录

**反馈提示：**
- 打开时：淡入动画（200ms）
- 关闭时：淡出动画（200ms）
- 复制成功：Toast"已复制"
- GIF加载中：显示加载动画

#### 边界情况处理
| 场景描述 | 触发条件 | 系统行为 | 提示信息 |
|---------|---------|---------|---------|
| GIF文件过大 | GIF文件>10MB | 显示首帧，提示性能模式 | "文件较大，已暂停播放" + "点击播放"按钮 |
| GIF帧率异常 | 帧率>60fps或<1fps | 限制播放帧率在1~60fps | 无 |
| 文件已删除 | 打开详情时文件不存在 | 显示错误提示，禁用操作按钮，并提供"重新定位"与"删除图片"按钮 | "原文件已丢失" |
| 图片尺寸过大 | 宽/高>5000px | 缩小至窗口内，提供"查看原图"按钮 | 无 |
| 无上/下一张 | 已是第一张或最后一张 | 禁用对应方向键 | 无 |

#### 算法说明
**GIF播放控制逻辑：**

```javascript
// 自动播放判断
if (fileFormat === 'GIF') {
  if (fileSize > 10MB) {
    autoPlay = false; // 大文件默认不播放
    showPlayButton = true;
  } else {
    autoPlay = true;
    loop = true; // 循环播放
  }
}

// 帧率限制
const targetFPS = Math.min(Math.max(originalFPS, 1), 60);
```

**图片缩放策略：**

```javascript
// 计算显示尺寸
const maxWidth = windowWidth * 0.8;
const maxHeight = windowHeight * 0.7;

if (imageWidth > maxWidth || imageHeight > maxHeight) {
  const scale = Math.min(maxWidth / imageWidth, maxHeight / imageHeight);
  displayWidth = imageWidth * scale;
  displayHeight = imageHeight * scale;
} else {
  // 原始尺寸显示
  displayWidth = imageWidth;
  displayHeight = imageHeight;
}
```
