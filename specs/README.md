# 当前 Spec

## 系统现状

当前产品可以先按下面这张心智图理解：

- 正式主路径只有两条：`搜索` 和 `图库管理`
- 搜索页承接三种状态：`冷启动首页`、`普通首页`、`非空搜索结果`
- 图库为空时，用户在搜索页内直接发起快速导入，不先跳去图库管理
- 图库管理负责导入后的整理、异常排查、失效清理和批量治理，不是首次使用默认必经页
- 私有角色库不是普通用户主路径，而是高级检索增强能力；当前实现仅在开发模式下通过图库管理高级能力区或开发路由进入
- 应用默认以侧边栏工作台启动；需要更大空间整理图片时，再进入展开态
- `清空图库`、调试、重建索引这类能力属于内部维护范畴，不属于普通产品功能

如果只想先快速建立全局认知，读到这里就够了。
