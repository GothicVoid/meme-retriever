# 系统总览

- 状态：approved
- 当前性：当前有效

## 当前结构

系统当前按以下层次组织：

1. 搜索主路径：空查询首页、冷启动首页、非空搜索结果
2. 图库管理路径：导入后的整理、修复、排查和批量治理
3. 设置路径：基础配置、窗口偏好、受控高级入口
4. 私有角色库：受控增强能力，不是普通主路径

## 文档分层

## 当前关键数据流

### 搜索

用户输入查询 -> 搜索主路径选择与排序 -> 结果展示 -> 复制 / 详情 / 失败修复

### 导入

用户从冷启动首页或图库管理发起导入 -> 统一导入协议 -> 索引构建 -> 图库与首页状态更新

### 窗口

用户切换页面或模式 -> 前端写入当前工作态 -> 后端应用窗口布局并保存偏好
