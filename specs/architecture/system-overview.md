# 当前系统快照

- 状态：approved

## 一眼看现状

当前系统不是“多页面桌面后台”，而是一个聊天伴随型找图工具。

核心结构只有两条正式主路径：

- `搜索`：默认入口，负责冷启动、普通首页和非空搜索结果
- `图库管理`：负责导入后的整理、排查、修复和批量治理

补充边界：

- `私有角色库` 是高级检索增强能力，不是普通用户主路径；当前实现仅在开发模式下暴露维护入口
- `清空图库`、调试、重建索引属于内部维护能力，不属于普通产品功能
- 应用始终从侧边栏态启动，需要更大工作区时才进入展开态

## 系统结构图

```text
表情包检索器
├── 搜索工作台（默认入口，sidebar）
│   ├── 冷启动首页：图库为空，先导入
│   ├── 普通首页：图库非空，继续使用
│   └── 搜索结果页：查询非空，找图与复制
├── 图库管理（expanded 优先）
│   ├── 全部图片
│   ├── 导入结果承接：查看失败项 / 查看本次新增
│   ├── 已发现失效图片入口（按需出现）
│   ├── 导入入口：导入图片 / 导入文件夹
│   └── 治理动作：删除、详情、失效清理
├── 开发模式补充入口
│   ├── 图库管理 > 高级能力 > 角色搜索增强
│   ├── /private-role-maintenance
│   └── 搜索页开发工具浮层
└── 内部维护能力
    ├── 清空图库
    ├── 调试搜索叠层
    └── 手动重建索引
```

## 页面与状态快照

| 页面 / 状态 | 触发条件 | 用户看到什么 | 主动作 |
|------------|---------|-------------|-------|
| 冷启动首页 | 图库为空且查询为空 | 导入引导、示例词、示例图区、搜索框 | `导入图片` |
| 普通首页 | 图库非空且查询为空 | 最近常用 / 最近使用、搜索框 | 继续找图或直接复制 |
| 搜索结果页 | 查询非空 | 结果列表、排序理由、失败反馈 | 复制、预览、详情 |
| 图库管理 | 用户主动进入治理路径 | 全部图片、导入结果承接、按需出现的失效治理入口 | 导入、删除、修复 |
| 私有角色库 | 开发模式下从图库管理高级能力区或开发路由进入 | 角色卡片维护与校验 | 创建、编辑、测试 |

## 当前主链路

### 1. 首次使用

```text
打开应用
  -> 侧边栏态启动
  -> 冷启动首页
  -> 导入图片
  -> 导入完成
  -> 开始搜索
  -> 复制图片
```

### 2. 日常使用

```text
打开应用
  -> 普通首页或直接聚焦搜索
  -> 输入台词 / 角色 / 动作 / 场景
  -> 查看结果
  -> 单击复制 / 双击详情
```

### 3. 搜索失败后的修复路径

```text
搜索失败或结果不理想
  -> 去图库管理
  -> 查看本次新增 / 查看失效图片
  -> 补充导入、删除记录、处理已发现问题
  -> 通过图库页头部返回搜索继续找图
```

## 当前规则

### 搜索

- 按记忆线索搜索，不要求用户维护文件名或标签
- OCR 和语义是主路，标记和热度只做增强
- 默认只展示高相关和较相关结果
- 排序解释使用用户能理解的自然语言

### 导入

- 冷启动首页用单按钮 `导入图片`，经轻弹层分流到 `选择图片 / 选择文件夹`
- 图库管理继续保留 `导入图片 / 导入文件夹` 双按钮
- 图片导入和文件夹导入共用一套导入状态机和索引任务管线
- 存储策略是路径引用，不复制原文件
- 搜索页右下始终保留 `图库` 入口；开发模式下额外显示开发工具浮层

### 窗口

- 应用始终从 `sidebar` 启动
- 进入图库管理或需要更大工作区时切到 `expanded`
- 不向普通用户暴露复杂窗口模式或默认打开方式设置

## 前后端边界

### 前端负责

- 页面职责分层与交互文案
- 搜索主路径与图库管理的路由和状态组织
- 导入入口分流与进度展示
- 侧边栏默认启动与图库管理展开态的布局请求触发

### 后端负责

- 图片导入协议的统一接收、展开、过滤和总数统计
- OCR、向量、自动提示词等索引构建
- 窗口布局应用与侧边栏快照持久化
- 私有角色库运行时读取与相关检索能力

### 约束

- 前端不维护两套导入状态机
- 普通用户路径不依赖独立设置模块或手动重建索引
- 私有角色库维护能力与普通用户路径分层

## 关键状态与事件

### 前端关键状态

- `navigation_state`：当前一级页面
- `windowMode`：当前工作态
- `gallery_view_mode`：图库管理当前视图
- `recent_searches`：搜索历史
- 冷启动导入状态：`未导入 / 选择中 / 导入中 / 导入完成`

### 关键事件

- 搜索框聚焦与发起搜索
- 首页快速导入动作
- 图库管理视图切换
- 窗口布局应用与偏好保存
- 高级入口曝光与点击

### 设计要求

- 导入入口可以分流，但导入执行状态必须统一
- 窗口模式变化要有唯一来源，避免“启动偏好”和“当前工作态”互相打架
- 模块文档描述长期边界，feature 文档描述具体行为和约束

## 哪些不在普通用户路径里

| 能力 | 当前定位 | 是否普通用户默认可见 |
|------|---------|--------------------|
| 私有角色库 | 高级检索增强能力 | 否，且当前仅开发模式可见 |
| 清空图库 | 内部重置能力 | 否 |
| 调试搜索 | 开发 / 维护工具 | 否，仅开发模式浮层可见 |
| 重建索引 | 维护工具 / 升级兜底能力 | 否，仅开发模式浮层可手动触发 |
