# 系统总览

- 状态：approved
- 当前性：当前有效

## 当前结构

系统当前按以下层次组织：

1. 搜索主路径：空查询首页、冷启动首页、非空搜索结果
2. 图库管理路径：导入后的整理、修复、排查和批量治理
3. 私有角色增强路径：图库管理中的高级检索增强能力
4. 内部维护路径：调试搜索、索引维护和数据重置

## 当前关键数据流

### 搜索

用户输入查询 -> 搜索主路径选择与排序 -> 结果展示 -> 复制 / 详情 / 失败修复

### 导入

用户从冷启动首页或图库管理发起导入 -> 统一导入协议 -> 索引构建 -> 图库与首页状态更新

### 窗口

应用启动或进入图库管理 -> 前端请求对应窗口布局 -> 后端应用窗口布局并保存侧边栏快照
