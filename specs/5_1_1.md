#### 修订记录

#### 功能描述
提供面向懒人用户的记忆式搜索输入框。用户无需维护文件名或标签，只需凭记忆输入图上的字、角色名、动作、表情或场景描述，系统即可自动选择合适的检索主路并返回结果。

#### 核心记忆线索
- 图上的字或台词
- 角色或主体名称
- 动作、表情、姿态
- 场景、情绪、氛围
- 少量用户主动维护的私有标记

#### 流程说明
1. 用户在搜索框中输入查询文本（中英文混合）
2. 系统检测输入停顿 300ms 后自动触发检索
3. 检索引擎分析查询意图，选择当前查询的主路
4. 系统融合主路相关性、辅路补充和轻量热度修正
5. 返回按相关性排序的结果列表
6. 用户可点击清空按钮清除输入

#### 交互说明
**页面元素：**
- 搜索框：占据页面顶部，支持自然语言输入
- 搜索图标：位于输入框左侧
- 清空按钮：输入内容后显示在右侧，点击清空输入

**交互行为：**
- 页面加载时自动聚焦搜索框
- 支持中英文、标点符号输入
- 实时 Debounce，停顿 300ms 后触发检索
- 按 Esc 键清空搜索内容
- 按 Ctrl/Cmd + F 重新聚焦搜索框

**反馈提示：**
- 输入时无加载状态（Debounce 期间）
- 检索开始后显示加载动画（如果耗时 >100ms）
- 无结果时显示："没找到相关图片，试试别的描述，比如台词、角色或动作？"

#### 搜索策略说明

搜索排序采用**动态主路 + 辅路补充 + 轻量热度修正**机制，而不是向用户暴露固定权重调参。

**总原则：**
- 先判断当前查询最像哪类记忆线索
- 由最可信的主路决定结果方向
- 其他信号只做补充，不稀释强命中
- 热度仅用于轻量重排，不压过明显相关结果

#### 主路定义

| 主路 | 典型查询 | 主作用 |
|-----|---------|-------|
| OCR 主路 | "你礼貌吗"、"别骂了" | 优先匹配图中文字或台词 |
| 语义主路 | "无语捂脸"、"跪地求饶" | 优先匹配动作、表情、场景、氛围 |
| 私有角色主路 | "阿布撇嘴"、"老板冷笑" | 先限定角色主体，再按动作或表情细排 |

#### 排序原则
建议以策略描述为准，不在产品层暴露固定线性权重：

```text
FinalScore = MainSignal + AuxSignal + PopularityBonus
```

其中：
- `MainSignal`：当前查询命中的主路得分
- `AuxSignal`：非主路信号提供的补充信息
- `PopularityBonus`：基于使用频次或最近使用的轻量修正

补充约束：
- 非空查询下，`PopularityBonus` 仅做轻量修正，建议影响不超过最终排序的 10%
- OCR 主路下，明确文本强命中结果优先于仅语义相似结果
- 语义主路下，允许无 OCR、无标记结果进入排序，但不能被高热度弱相关结果压过

#### 主路选择规则
- 当查询明显包含可匹配的台词、短句、字幕片段时，优先走 OCR 主路
- 当查询命中私有角色名或角色别名时，优先走私有角色主路
- 当查询主要描述动作、表情、场景或氛围，且无明显 OCR / 角色命中时，优先走语义主路
- 手动标记和自动提示词不单独成为主路，只作为增强信号

#### 私有角色搜索说明
当查询命中某个私有角色名或别名时：
1. 系统先根据该角色的示例图 embedding 召回“像这个角色”的候选图片
2. 再使用完整查询中的动作、表情、场景描述做细排
3. 若查询仅包含角色名，则主要按角色相似度排序
4. 若角色缺少示例图，则降级为普通语义搜索，并保留弱角色文本命中

示例：

```text
查询："阿布撇嘴"

角色约束：
- 命中私有角色 "阿布"
- 用阿布的示例图先召回同角色候选

动作细排：
- 保留 "撇嘴" 语义用于区分同角色多张图片

预期结果：
- 先稳定召回阿布
- 再让更接近撇嘴动作的图片排前
```

#### 结果过滤规则
| 场景 | 判断条件 | 处理规则 |
|-----|---------|---------|
| 空查询 | 用户清空搜索框或初次进入 | 展示常用或最近使用图片 |
| 纯语义命中 | 无 OCR / 标记命中，但语义相似度达标 | 允许进入排序结果 |
| 文本弱匹配 | 仅 OCR / 标记弱命中，且质量不足 | 直接过滤不展示 |
| 私有角色命中 | 命中角色名但示例图缺失 | 降级为普通语义搜索 |

说明：
- 不再使用统一固定相关性阈值过滤所有结果
- 不再要求用户手动调整 OCR / 标签 / 语义的权重

#### 边界情况处理
| 场景描述 | 触发条件 | 系统行为 | 提示信息 |
|---------|---------|---------|---------|
| 输入为空 | 用户清空搜索框或初次进入 | 展示常用或最近使用图片 | 无提示 |
| 无匹配结果 | 无图片通过主路或辅路过滤规则 | 展示空状态页面 | "没找到相关图片，试试别的描述，比如台词、角色或动作？" |
| 输入过长 | 输入字符 > 200 | 只取前 200 字符进行检索 | 无提示（静默处理） |
| 特殊字符 | 输入包含 emoji 等特殊符号 | 自动过滤无效字符并保留可用内容 | 无提示 |
| 图库为空 | 用户未添加任何图片 | 展示引导页 | "还没有图片哦，点击添加开始使用吧" |
| 冷门角色 | 模型不认识角色，但用户已配置私有角色 | 走私有角色主路 | 无提示 |
| 冷门角色未配置 | 模型不认识角色，私有角色库也无配置 | 降级为普通语义搜索 | 无提示 |

#### 权限说明
- 可访问角色：所有用户
- 权限校验规则：无需校验

#### 扩展点说明
- 支持搜索历史记录（下拉展示）
- 支持基于私有标记和角色的快捷筛选
- 支持更细的搜索模式预设（如文字优先、语义优先）
