### 5.1.1 搜索输入功能（优先级：P0）

#### 功能描述
提供智能搜索输入框，支持用户通过多种方式描述目标图片，自动触发语义检索。

#### 流程说明
1. 用户在搜索框中输入查询文本（中英文混合）
2. 系统检测输入停顿300ms后自动触发检索
3. 检索引擎分析查询意图，匹配图库中的图片
4. 返回按相关性排序的结果列表
5. 用户可点击清空按钮清除输入

#### 交互说明
**页面元素：**
- 搜索框：占据页面顶部，圆角矩形设计
- 搜索图标：位于输入框左侧
- 清空按钮：输入内容后显示在右侧，点击清空输入

**交互行为：**
- 页面加载时自动聚焦搜索框
- 支持中英文、标点符号输入
- 实时Debounce，停顿300ms后触发检索
- 按Esc键清空搜索内容
- 按Ctrl/Cmd + F重新聚焦搜索框

**反馈提示：**
- 输入时无加载状态（Debounce期间）
- 检索开始后显示加载动画（如果耗时>100ms）
- 无结果时显示：「没找到相关图片，试试其他描述？」

#### 算法说明

检索排序采用**相关性+热度**双因子融合机制：

**最终得分公式：**
```
Final_Score = α·Relevance + β·Popularity
```

**权重配比：**
- α（相关性权重）= 0.75
- β（热度权重）= 0.25

---

**一、相关性得分 Relevance**

采用三路融合策略，加权求和计算相关性得分：

```python
Relevance = w1 · S_keyword + w2 · S_ocr + w3 · S_clip
```

**三路得分计算规则：**

| 路径 | 计算方式 | 得分范围 |
|-----|---------|---------|
| S_keyword（实体匹配） | 完全匹配=1.0<br>部分匹配=0.8<br>关联标签=0.5 | 0~1.0 |
| S_ocr（文字匹配） | 字符覆盖率 <br>覆盖率 = len(query ∩ ocr_text) / len(query) | 0~1.0 |
| S_clip（语义相似度） | (cosine_similarity + 1) / 2 | 0~1.0 |

**三路权重配置：**
```
默认值：w1 = 0.3, w2 = 0.4, w3 = 0.3
约束条件：w1 + w2 + w3 = 1
```
（支持在设置页面手动调整，下次查询生效。调整时自动归一化确保总和为1）

---

**二、热度得分 Popularity**

基于使用频次的对数归一化：

```python
freq_score = log(1 + click_count) / log(1 + max_click)
```

对数化处理避免高频图片过度集中，给新图片更多曝光机会。

---

**三、结果过滤规则**

| 场景 | 判断条件 | 处理规则 |
|-----|---------|---------|
| 冷启动（新图） | use_count = 0 | popularity = 0.1（给予较低初始值） |
| 纯语义命中 | `cosine_similarity >= 0.15` 且无 OCR/标签命中 | 允许进入排序结果 |
| 文本弱匹配 | 仅 OCR/标签命中，且 `S_ocr < 0.2` 且 `S_keyword < 0.5` | 直接过滤不展示 |

说明：结果过滤不再使用统一 `relevance < 0.2` 阈值。语义路径与文本路径分别判定，避免纯语义召回在融合时被误过滤。

---

**计算示例：**

```
查询："你礼貌吗"

图片A：
- S_keyword = 0（无标签匹配）
- S_ocr = 1.0（完整包含"你礼貌吗"）
- S_clip = 0.85
- Relevance = 0.3×0 + 0.4×1.0 + 0.3×0.85 = 0.655
- click_count = 50, max_click = 200
- Popularity = log(51)/log(201) = 0.73
- Final_Score = 0.75×0.655 + 0.25×0.73 = 0.674

图片B：
- S_keyword = 0.5（关联标签"质疑"）
- S_ocr = 0.3（包含"你"）
- S_clip = 0.92
- Relevance = 0.3×0.5 + 0.4×0.3 + 0.3×0.92 = 0.576
- click_count = 0（新图）
- Popularity = 0.1（冷启动默认值）
- Final_Score = 0.75×0.576 + 0.25×0.1 = 0.457

结果：图片A排在前面（综合考虑相关性和热度）
```

#### 边界情况处理
| 场景描述 | 触发条件 | 系统行为 | 提示信息 |
|---------|---------|---------|---------|
| 输入为空 | 用户清空搜索框或初次进入 | 展示使用频次最高的9张图片 | 无提示 |
| 无匹配结果 | 无图片通过语义或文本过滤规则 | 展示空状态页面 | "没找到相关图片，试试其他描述？" |
| 输入过长 | 输入字符>200 | 只取前200字符进行检索 | 无提示（静默处理） |
| 特殊字符 | 输入包含表情符号、emoji等 | 自动过滤特殊字符 | 无提示 |
| 图库为空 | 用户未添加任何图片 | 展示引导页 | "还没有图片哦，点击添加开始使用吧" |

#### 权限说明
- 可访问角色：所有用户
- 权限校验规则：无需校验

#### 数据埋点
| 埋点事件 | 触发时机 | 采集参数 | 用途说明 |
|---------|---------|---------|---------|
| search_query | 用户输入触发检索时 | query_text, result_count, response_time | 分析搜索词分布与性能 |
| search_empty_result | 检索无结果时 | query_text | 优化检索算法 |
| search_clear | 用户点击清空按钮 | query_text | 分析用户行为 |

#### 扩展点说明

- 支持智能联想（基于已有标签和OCR文本）
- 支持高级搜索语法（如AND/OR/NOT）
