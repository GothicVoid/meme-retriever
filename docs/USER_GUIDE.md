# 使用说明

本文档面向直接安装 release 安装包的最终用户，说明这个应用怎么开始用、平时怎么用，以及哪些功能属于进阶能力。

## 这是什么

`meme-retriever` 是一个本地表情包检索工具。

你可以把它理解为一个专门管理和搜索本地表情包的小型工作台，适合解决这类问题：

- 表情包都在本地文件夹里，但越来越难找
- 想通过关键词或一句描述，快速找到想发的图
- 想把图库里的无效图片、重复图片、不想要的图片整理掉

## 第一次打开后该做什么

第一次使用时，先完成一件事：

- 导入你的本地图片

如果当前图库还是空的，搜索页不会直接给出有效结果。  
先把常用表情包所在的文件夹导入进来，应用建立索引后，后续搜索才会更快、更准。

## 日常怎么用

大多数情况下，你只需要理解两个入口：

- `搜索`
- `图库管理`

推荐使用顺序：

1. 先导入本地图片
2. 在搜索页输入关键词或更接近语义的描述
3. 从结果里挑选你要的表情包
4. 如果图库需要整理，再进入图库管理页处理

## 搜索

搜索页是最常用的入口，适合“我现在就想找到一张图”这种场景。

你可以在这里做的事情包括：

- 输入关键词找图
- 输入更自然的描述找图
- 在结果不理想时改写描述继续搜索
- 图不够时继续补充导入

如果你平时主要是“找图然后发图”，大部分时间都会停留在这里。

## 图库管理

图库管理适合“整理图库”，不是“快速找图”。

你可以在这里做的事情包括：

- 查看当前已经导入的图片
- 删除不需要的图片
- 清理失效内容
- 排查导入后不正常的内容
- 做批量整理

如果你的目标是把图库维护干净，而不是立刻找一张图，应该优先进入这里。

## 进阶能力

部分能力属于进阶能力，不是普通用户的默认主路径，例如：

- 标记增强
- 私有角色库维护

这些能力更适合在你已经熟悉基本搜索和图库管理之后，再按需使用。

## 需要知道的边界

- 这是一个本地优先工具，不是在线图库
- 搜索效果依赖你已经导入的本地图片
- 图库为空时，应先导入图片，再开始搜索
- 某些维护或调试能力不属于普通日常使用范围
