# PRD：表情包检索器

**版本** v0.1 | **状态** 草稿

---

## 一、产品概述

### 背景与目标

聊天场景中，用户往往知道自己想发哪张表情包，却难以在本地图库中快速定位。本产品通过语义检索，让用户用自然语言描述即可秒找目标图片，降低"翻图"成本。

### 产品形态

桌面端本地应用（优先），图库与索引数据均存储在本地，不依赖网络。

### 成功指标

- 检索响应时间 < 500ms
- 用户找到目标图片的平均操作次数 ≤ 2 次

---

## 二、目标用户

| 用户类型 | 描述 | 核心诉求 |
|---|---|---|
| 重度表情包用户 | 本地存有大量图片，日常高频使用 | 快速定位、快速复制 |
| 内容创作者 | 需要在创作中引用特定梗图 | 按出处/角色精准检索 |
| 轻度用户 | 图库较小，偶尔使用 | 简单易上手 |

---

## 三、功能需求

### 3.1 检索页（主界面）

#### 搜索输入

- 支持中英文混合输入
- 实时检索，输入停顿 300ms 后触发（debounce）
- 支持清空按钮
- 支持以下查询类型，系统自动识别，无需用户手动切换：

| 查询类型 | 示例 |
|---|---|
| 用户自定义标签 | `搞笑` `工作` |
| 台词 / 配文 | `你礼貌吗` |
| 梗 / 名场面 | `蚌埠住了` |
| 内容描述 | `一只猫在打字` |
| 角色名 / 昵称 | `派大星` |
| 出处 / 作品名 | `甄嬛传` |

#### 结果列表

- 无查询时：展示使用频次最高的图片，按频次降序排列
- 有查询时：展示语义得分最高的图片，按得分降序排列
- 默认展示 9 张，可在设置中调整（范围 1 ~ 21，步长 1）
- 提供「展示更多」入口，最多展示 21 张，超出不再加载
- 空结果时展示提示文案：「没找到相关图片，试试其他描述？」

#### 图片交互

| 操作 | 行为 |
|---|---|
| 左键单击 | 复制图片到系统剪贴板，给出轻提示（Toast） |
| 长按拖动 | 支持拖入聊天输入框等可接受图片的区域 |
| 右键菜单 → 删除 | 二次确认后从图库中删除 |
| 右键菜单 → 编辑标签 | 打开标签编辑弹窗（见 3.3） |
| 右键菜单 → 在文件夹中显示 | 打开系统文件管理器并定位到原文件 |

#### 快捷键

| 快捷键 | 行为 |
|---|---|
| `Ctrl/Cmd + F` 或直接输入 | 聚焦搜索框 |
| `Esc` | 清空搜索框 |

---

### 3.2 图库管理

#### 添加图片

支持以下方式：

1. 点击「添加图片」按钮，打开系统文件选择器（支持多选）
2. 拖拽图片文件到图库管理页面
3. 从剪贴板粘贴（`Ctrl/Cmd + V`）

支持格式：JPG、PNG、GIF、WEBP

添加后自动触发索引构建，期间显示进度状态。

#### 删除图片

- 支持单张删除（右键菜单）
- 支持多选批量删除（Checkbox + 批量操作栏）
- 删除前二次确认，说明操作不可撤销
- 仅删除应用内记录，不删除本地原始文件（可在设置中切换此行为）

#### 筛选

- 筛选输入框与主界面检索逻辑一致，支持语义搜索（标签、台词、内容描述、角色名、出处等），结果按语义得分排序

#### 列表视图

- 默认视图：全部图片，按添加时间降序
- 分组视图：按日 / 周 / 月 / 年分组，组内按添加时间降序
- 视图切换保留用户上次选择

---

### 3.3 标签系统

- 每张图片可添加多个标签，标签为纯文本字符串
- 标签编辑弹窗支持：添加、删除单个标签
- 标签输入支持自动补全（基于已有标签）
- 标签参与检索权重，用户自定义标签优先级高于自动识别内容

---

### 3.4 单图详情

点击图片缩略图右键 → 「查看详情」，或在图库管理中点击图片进入：

- 大图预览
- 元数据：文件名、格式、尺寸、添加时间、使用次数
- 标签列表（可编辑）
- 操作：复制、删除

---

### 3.5 设置

| 配置项 | 默认值 | 说明 |
|---|---|---|
| 默认展示数量 n | 9 | 范围 1 ~ 21 |
| 删除图片时是否同时删除原文件 | 否 | — |
| 图库存储路径 | 应用数据目录 | 支持自定义 |
| 检索语言偏好 | 跟随系统 | 中文 / 英文 / 自动 |

---

## 四、非功能需求

| 类别 | 要求 |
|---|---|
| 性能 | 检索响应 < 500ms（图库 ≤ 5000 张） |
| 存储 | 索引文件大小合理，不超过图片总大小的 20% |
| 稳定性 | 添加图片失败时给出明确错误提示，不影响已有图库 |
| 隐私 | 所有数据本地存储，不上传任何图片或元数据 |
| 兼容性 | 支持 macOS 13+、Windows 10+ |
